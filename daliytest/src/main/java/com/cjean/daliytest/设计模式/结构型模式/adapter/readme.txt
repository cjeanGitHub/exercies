适配器模式（adapter）   
	定义：将一个雷的接口转换成客户希望的另外一个接口，Adapter模式是得原本由于
		接口不兼容而能一起工作的哪些类可以在一起工作。
	将被适配器/需要适配的类/Adaptee（键盘）的功能通过适配器（Adapter/转换产品）
	提供给目标接口（Target/例子中的电脑USB接口）
	
	模式中的角色:
		- 目标接口（Target-例子中的电脑USB接口）
			：客户所期待的接口，目标可以是具体的货抽象的类，也可以是接口
		- 需要适配的类（Adaptee/例子中的键盘）
			：需要适配的类/适配者类  提供功能的，但是接口不合适
		- 适配器（Adapter/提供转换功能的）
			：通过包装一个需要适配的对象，把原接口转换成目标接口
